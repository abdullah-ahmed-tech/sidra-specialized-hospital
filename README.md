# Sidra Specialized Hospital Platform

A commercial-grade hospital platform designed for showcase, client delivery, and future expansion.

## Products
- Patient Mobile App
- Official Hospital Website
- Admin Dashboard
- Central Backend API

## Stack
- Expo React Native
- Next.js
- NestJS
- PostgreSQL
- Prisma
- Tailwind CSS

## Core Features
- doctor directory
- departments
- services
- appointment booking
- admin operations
- dashboard metrics
- role-ready backend architecture

## Monorepo Structure
- apps/mobile
- apps/website
- apps/admin
- apps/api
- packages/ui
- packages/types
- packages/utils
- packages/config

## Run
```bash
pnpm install
pnpm dev:api
pnpm dev:admin
pnpm dev:website
pnpm dev:mobile
